
# Lambda Equivalent Media Uploader to Social Platform

This repository embodies the component of Microservice which play role to upload the media file's from SQS to defined Social Media.

By implementing worker pool pattern, this project can handle **Batch** data from **SQS** at a time. 


SQS will play role to send message Event and depending on settings SQS will triger AWS lambda function to execute events,
Depending on messages / Batch Messages.
